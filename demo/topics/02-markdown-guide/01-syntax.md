# Markdown Syntax

A quick reference for what the wiki parser supports.

## Headings

```
# H1
## H2
### H3
```

## Text

**Bold text** with `**double asterisks**`.

Inline `code` with backticks.

## Lists

Unordered:
- Item one
- Item two
- Item three

Ordered:
1. First
2. Second
3. Third

## Code blocks

```python
def hello():
    print("Hello, wiki!")
```

## Callouts

> This is an info callout — use `> text`

>! This is a warning callout — use `>! text`
